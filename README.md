# Praca Inżynierska - Szeregi Czasowe
Aplikacja porównująca szeregi czasowe i wizualizująca je na wykresie. Powstała w ramach pracy inżynierskiej na zlecenie firmy Rockwool.

## Uruchamianie aplikacji
Aplikacja jest budowana za pomocą Docker Compose. W celu zbudowania i uruchomienia wpisujemy w terminalu komendę: `docker compose up --build`.
