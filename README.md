# Praca Inżynierska - SeriesDiff
Aplikacja służąca do porównywania szeregów czasowych i wizualizacji ich na wykresie. Powstała w ramach pracy inżynierskiej na zlecenie firmy Rockwool.


## Uruchamianie aplikacji
Aplikacja w wersji developerskiej jest dostępna pod adresem www-dev.seriesdiff.com.
Dostęp do API jest możliwy pod adresem api-dev.seriesdiff.com.

Aplikacja może być budowana lokalnie   za pomocą Docker Compose. W celu zbudowania i uruchomienia aplikacji potrzebujemy sklonować repozytorium komendą `git clone`. Po sklonowaniu aplikacji wchodzimy w folder główny projektu i wpisujemy w terminalu komendę: `docker compose up --build`.
